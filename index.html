O que são Seletores de Especificidade

Os seletores de especificidade no CSS são mecanismos que determinam a prioridade de aplicação da estilização em elementos HTML. Eles dizem respeito ao quão específico você está sendo ao estilizar algo no seu código. Ajudam a definir qual estilo deve ser aplicado a um elemento específico, levando em consideração a força e a hierarquia dos seletores.

Para que servem os seletores de especificidade?

Os seletores de especificidade no CSS são mecanismos que determinam a prioridade de aplicação da estilização em elementos HTML. Eles dizem respeito ao quão específico você está sendo ao estilizar algo no seu código. Ajudam a definir qual estilo deve ser aplicado a um elemento específico, levando em consideração a força e a hierarquia dos seletores.

Principais Tipos de Seletores, dos menos aos mais específicos:

Seletores de elementos e pseudo-elementos:

Exemplo: p, div, h1, etc.

São os seletores mais genéricos, selecionam elementos com base em seu tipo (tag). Têm a menor especificidade.

Seletores de classe, atributo e pseudo-classes:

Exemplo: .classe

Selecionam elementos com base em uma classe atribuída. Têm uma especificidade maior do que os seletores de tipo.

Seletores de ID:

Exemplo: #id

Selecionam elementos com base em um ID específico atribuído. Têm uma especificidade ainda maior do que os seletores de classe.

como são calculados

Os seletores possuem um número que define a sua prioridade e importância antes de ser considerado à aplicação. O cálculo de especificidade é uma combinação desses três tipos de seletores, que gera algo mais ou menos assim:

(0. 0. 0)

Esses números representam, respectivamente a quantidade de ID’s, classes e elementos no arquivo CSS. Os números a esquerda sempre terão mais prioridade que os da direita, então os números são analisados para assim determinar qual o seletor mais específico (terá mais prioridade) no momento da aplicação.

!Important Value

O CSS disponibiliza uma forma de "burlar" os níveis de especificidade através da palavra-chave !important

O estilo que for marcado juntamente com essa palavra-chave terá prioridade acima dos outros seletores, mas respeitando o comportamento de cascata mencionado anteriormente.
